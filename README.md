# 🧑‍💼 Employee Management System — SQL Project

This project represents a complete end-to-end **relational database design and SQL analytics case study** built on an Employee Management dataset.

The objective is to simulate real-world HR operations including:

* Employee data management
* Department and job role structuring
* Salary and bonus tracking
* Qualification management
* Leave and attendance monitoring
* Payroll processing and reporting

---

## 📌 Project Overview

Modern organizations generate complex workforce data across multiple departments.

This project focuses on:

* Designing a **fully normalized relational database**
* Establishing **primary & foreign key relationships**
* Implementing **data integrity constraints**
* Writing analytical SQL queries for HR insights
* Performing payroll and workforce analysis

The database is implemented using **MySQL**.
